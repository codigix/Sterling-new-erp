SYSTEM REQUIREMENTS FOR VPS DEPLOYMENT
================================================================================

Required:
  • Linux OS (Ubuntu 18.04+, CentOS 7+, Debian 9+)
  • Node.js 14.0.0 or higher
  • npm 6.0.0 or higher
  • MySQL 5.7 or higher
  • Nginx 1.14+ OR Apache 2.4+
  • 2GB RAM minimum (4GB recommended)
  • 10GB disk space minimum (20GB recommended)

Optional but Recommended:
  • Let's Encrypt for SSL certificates
  • PM2 for process management
  • UFW firewall
  • Fail2ban for security

================================================================================
QUICK START DEPLOYMENT
================================================================================

1. PREPARE VPS:
   sudo apt update && sudo apt upgrade -y
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt install -y nodejs nginx mysql-server git
   sudo npm install -g pm2

2. CLONE PROJECT:
   cd /var/www
   git clone your-repo-url
   cd Sterling-erp

3. SETUP ENVIRONMENT:
   cp .env.production.example .env
   nano .env  (edit with your production values)

4. AUTO-DEPLOY:
   sudo chmod +x deploy.sh
   sudo ./deploy.sh

5. CONFIGURE WEB SERVER:
   For Nginx: sudo cp nginx.conf.example /etc/nginx/sites-available/sterling-erp
   For Apache: sudo cp apache.conf.example /etc/apache2/sites-available/sterling-erp.conf
   (Edit domain name and paths in configuration)

6. SETUP SSL:
   sudo certbot --nginx -d your_domain.com (Nginx)
   OR
   sudo certbot --apache -d your_domain.com (Apache)

7. VERIFY DEPLOYMENT:
   curl https://your_domain.com/api/health
   pm2 status
   pm2 logs sterling-backend
